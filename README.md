# To-Do List App

A clean, modern, and minimalist web-based to-do list application with data persistence using localStorage.

## Features

- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Task counter (total and completed)
- Data persistence using localStorage (JSON format)
- Responsive design
- Clean, modern UI with smooth animations

## Getting Started

### Local Development

Simply open `todo-app.html` in any modern web browser (Chrome, Firefox, Edge, Safari).

No build process or dependencies required!

### Deployment

This app can be deployed to Vercel, Netlify, GitHub Pages, or any static hosting service.

## Technical Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with Flexbox, transitions, and gradients
- **Vanilla JavaScript**: No frameworks or libraries
- **localStorage**: Client-side data persistence with JSON
